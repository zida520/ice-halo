1.创建运行环境，（即使没有内核，也要记得在特殊项目上创建一个虚拟环境）
如果是我，会创建一个requirements.txt，写上安装的库和注意事项
但是没有只能不断的试错，（错误有50%是因为没有安装库导致无法识别关键字，尤其是数据分析和机器学习[这两个东西还真般配]，代码是一格一格的）
anaconda的妙处是针对不同的项目创建不同的环境

创建环境，
python -m venv venv
venv\Scripts\activate

根据技术栈安装的库
pip install pandas pymysql sqlalchemy pyecharts
pandas      数据清洗装换用
pymysql      负责链接并执行sql语句
sqlalchemy  sql工具包，配合 pandas 的 to_sql 和 read_sql 方法操作 MySQL。
pyecharts    用于生成交互式图表

同时还有jieba，生成报告日志
wordcloud，兼容词云

pip install -r requirements.txt
################
示例 requirements.txt 内容
pandas==2.0.3
pymysql==1.1.0
sqlalchemy==2.0.19
pyecharts==2.0.4

2.程序文件的修改
data = pd.read_csv(r'D:\codebox\0shuju\倒闭企业分析\com.csv')
engine = create_engine('mysql+pymysql://root:xxxxxx%40SQL@localhost:3306/company_db')
data是解释文件位置
engine是链接mysql
同时由于mysql服务器密码有@，要改成 URL 编码格式：%40
3.将虚拟环境注册为内核
pip install ipykernel
python -m ipykernel install --user --name=myenv --display-name="我的项目环境"
这个就可以使用
python -m ipykernel install --user --name=deathcompany --display-name="倒闭企业分析环境"

4.
启动mysql的服务器
net start mysql80

登录 MySQL 创建一个数据库
mysql -u root -p
创建数据库
CREATE DATABASE company_db;
退出 MySQL：
EXIT

环境总结
需要库，数据集和地址，链接mysql并创建一个库，注册成内核
